# 2020 2학기 자료구조
  ### 사용되는 여러 자료구조를 c++을 사용하여 직접 구현
* ArrList, LinkedList
  + 기본적인 array 구현
  + singleLinkedList 구현
* ArrStack
  + array기반 stack구현
* linkedList
  + singleLinkedList data 삭제부분 구현
* queue
  + array기반 queue구현
  + linkedList기반 queue구현
  + 위 두가지 queue를 통해 각각 간단한 미니게임 구현
* sequenceList
  + linkedList기반 sequence구현
* Tree
  + vector를 사용한 Tree구현
  + 해당 Tree의 Depth표현
* TreeTraversal
  + vector를 사용한 Tree의 postOrder, preOrder구현
